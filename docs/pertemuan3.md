# Block, Inline, dan Inline-Block

## Latar Belakang

Display di CSS dipakai untuk ngatur cara elemen HTML ditampilkan di halaman. Ada tiga yang sering dipakai: block, inline, dan inline-block. Elemen block seperti <div> akan memenuhi satu baris penuh dan bisa diatur ukuran (width, height), cocok buat layout besar. Elemen inline seperti <span> tampil dalam satu baris bersama elemen lain dan gak bisa diatur ukurannya, biasanya dipakai buat teks atau elemen kecil dalam paragraf. Pemilihan display ini penting supaya layout web kita rapi dan sesuai kebutuhan.
